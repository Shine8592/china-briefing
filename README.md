# 中文资讯简报技能

多尺度中文资讯简报，从一条街道到全球，任意地理范围均可生成。

## 技能列表

### 📰 china-briefing

从主流媒体、街道公众号、社交平台获取资讯，结构化输出简报。

**核心特点：任意尺度**
- 全球 → 国家 → 省 → 市 → 区 → 街道 → 社区
- 每层自动匹配搜索源

**简报结构：**
核心结论 → 头条3条（含嘟嘟点评）→ 分类热点 → 数据看板 → 行动建议

**安装：** 在 ClawHub 搜索 `china-briefing`

## 🔐 API 密钥配置

### Tavily API

本技能使用 Tavily API 进行高质量实时新闻搜索。

**获取密钥：**
1. 访问 [Tavily 控制台](https://app.tavily.com/)
2. 注册/登录账户
3. 获取 API 密钥（免费套餐：1000次/天）

**配置方法（任选其一）：**

#### 方法1：环境变量（推荐）
```bash
export TAVILY_API_KEY="your-api-key-here"
```

#### 方法2：配置文件
```bash
cp config.example.yaml config.yaml
# 编辑 config.yaml 填入您的密钥
```

### 🛡️ 安全提醒

**⚠️ 重要：**
- 不要将 API 密钥提交到公开仓库
- 定期轮换密钥（建议每3个月）
- 监控 API 使用情况
- 如果密钥泄露，立即在 Tavily 控制台撤销
